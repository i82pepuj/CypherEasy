Cifrado Afín


Cifrado creado en 1967, el cual consiste en la sustitución siguiendo la siguiente fórmula matemática:

  • Ci = (a*mi+b) mod n
  • Mi = (Ci-b)*d mod n

    donde:

  • Ci: símbolo cifrado del texto
  • a: constante de decimación
  • mi: símbolo del texto a cifrar
  • b: constante de desplazamiento
  • n : número de símbolos del alfabeto de cifrado
  • Mi: símbolo descifrado
  • d: inverso de a modulo n

    Además, se debe satisfacer que:

  • 0 <= b <= 1
  • a != 0
  • mcd (a,n) = 1

    Por ejemplo, para el mensaje “hola” a = 3, b = 5 y n = 26.

    h = 7 => Ci = (7*3+5) mod 26 = 0 = a

    o = 14 => Ci = 21 = v

    l = 11 => Ci = 12 = m

    a = 0 => Ci = 5 = f

    Por lo tanto C = avmf.

    Para descifrar se averigua d, que en este caso d = 9 ya que d = 3,9 = 27 = 1 mod 26 y se aplica la fórmula de descifrado termino por término.

    a = 0 => Mi = (0-5)*9 mod 26 = 7 =h

    v = 21 => Mi = 14 = o

    m = 12 => Mi = 11 = l

    f = 5 => Mi = 0 = a

    M = “hola”, que es nuestro mensaje original. {% endblock %}
